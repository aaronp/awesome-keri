# Awesome KERI

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A curated index of resources for **KERI (Key Event Receipt Infrastructure)** and the ecosystem around **ACDC credentials, decentralized identity, and verifiable data registries**.

This repository aims to provide a **concise map of the KERI ecosystem** — linking to high-quality specifications, implementations, talks, and projects.

It is intentionally **minimal**: entries should be useful, reputable, and directly relevant.

Contributions welcome.
